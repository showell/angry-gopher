# Agent-collaboration docs

Durable notes about how Steve and his collaborators work with AI
coding agents (currently Claude Code). Not secret — these are the
conventions, not the active experiments.

Audience is split:

- **Humans** picking up the collaboration style for the first time →
  start with `ONBOARDING.md`.
- **Agents** joining a new human → start with `AGENT_CONVENTIONS.md`.
- **A human bootstrapping Claude Code** → `CLAUDE_SETUP.md`.

These docs live in Angry Gopher because Gopher is the longest-running
project in the suite. They're not Gopher-specific.

## Files

- `ONBOARDING.md` — for a human new to human-agent collaboration.
- `AGENT_CONVENTIONS.md` — for an agent joining a new collaborator.
- `CLAUDE_SETUP.md` — practical setup for a fresh Claude Code install.
- `PATTERNS.md` — reference sheet of conventions that recur across
  our projects.

## Context

Steve and Apoorva are the current collaborators. Steve has been
working with Claude Code intensively for weeks; Apoorva is in the
process of getting set up. The patterns here came out of that work.
They are opinions, not mandates — adopt what resonates, skip what
doesn't, document deviations.
